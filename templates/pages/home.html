{% trans "Easy to use, fast and lightweight operating system based on Free Software" %}

{% trans "Lightweight, fast and easy to use" %}

{% blocktrans %} Maui aims to offer an excellent desktop experience. Instead of forcing you to know every detail of your computer we expect you to turn it on, sit down and simply use it. {% endblocktrans %}

{% blocktrans %} In order to do so we have tailored a solution that can boot in less than 4 seconds to a nice, easy to use and flicker-free desktop environment. {% endblocktrans %}

{% trans "Coherent and integrated" %}

{% blocktrans %} All the applications shipped with Maui are designed to work together, are consistent and coherent because they use the same framework: Qt 5. Qt is the most advanced and modern application and UI framework available, brought to you by a community lead by Nokia and used by many companies. {% endblocktrans %}

{% blocktrans %} A lot of features of the desktop are also available to the applications, for example all the applications are capable of managing compressed archives without to implement this capability themselves. Also, applications can let other applications use some of their features without the need to implement it over and over. {% endblocktrans %}

{% trans "Stable and reliable" %}

{% blocktrans %} One of the problems with the existing Linux-based solutions is that they are fast moving targets. For us a stable API and ABI is indispensable, it avoids software breakage assuring third party developers their software will still work after an update of the operating system. {% endblocktrans %}

{% trans "Next-generation technologies" %}

{% blocktrans %} The graphics infrastructure is completely new, instead of the ancient X11 display server we use the faster and modern Wayland, which help us deliver a very pleseant and efficient windowing system. {% endblocktrans %}

{% blocktrans %} Maui is also completely integrated with systemd in order to bring up your desktop in a few seconds. {% endblocktrans %}

{% blocktrans %} We also follow and support the new trends like BTRFS and LLVM. {% endblocktrans %}

{% trans "Innovative file hierarchy" %}

{% blocktrans %} Maui also separates the file system hierarchy in three areas: system, common and user. Programs and libraries shipped by the operating system and their configuration files are located under the /system directory, while programs and libraries installed by the user are either under /common if available to all the users or under the user's home directory if used only by a single account. In case of problems you can always disable everything but system booting into a system {% endblocktrans %}

{% trans "Social desktop" %}

{% blocktrans %} Maui aims at complete integration with the mobile world and social media. An API to bring access to your social accounts to all the applications is in development. This means that you will be able to post a link on Facebook to the Web site your visiting, upload photos to Flickr and Facebook from your desktop, change your status on GoogleTalk and Facebook at the same time, access your GDrive files as they were into your hard drive, reach all your contacts no matter what IM network or social media they are on, and so on. {% endblocktrans %}
