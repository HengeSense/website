{% blocktrans %} Maui aims to offer an excellent desktop experience. Instead of forcing you to know every detail of your computer we expect you to turn it on, sit down and just use it. {% endblocktrans %}

{% blocktrans %} It's not the kind of Linux distribution you tried before. Instead of including a plethora of different desktop environments, Maui has only one made to overcome the boundaries of the current solutions. Hawaii the desktop environment, has a balanced settings policy, between the minimalist approach where almost no preferences are available and the highly-customizable approach where every single detail can be changed. Sane defaults for the needs of most users will make you feel at home. {% endblocktrans %}

{% blocktrans %} The "shell" is modular, this means we can create variants optimized for the device where you are running Maui. As the technology we are using will be widely adopted our applications and thirdparty's will have a different UX experience for both desktop and mobile devices. {% endblocktrans %}

{% blocktrans %} We pay attention to the details in order to offer you the most exciting user experience possible. {% endblocktrans %}

{% trans "Advanced Technology" %}
{% trans "Amazing and powerful technologies." %}


{% blocktrans %} Wayland is the most exciting piece of technology for the Linux graphics stack in years and Maui is the first Linux distribution to use it, in fact Maui couldn't be done without it. {% endblocktrans %}

{% blocktrans %} Wayland provides a method for compositing window managers to communicate directly with applications and video hardware, its goal is a system where every frame is perfect, where applications are able to control the rendering enough that users won't see tearing, lag, redraw or flicker. {% endblocktrans %}

{% blocktrans %} Qt is an application development framework based on C++ and has been chosen to implement the desktop environment and applications. Over 450,000 developers in more than 70 industries have chosen Qt to build advanced applications and devices. {% endblocktrans %}

{% blocktrans %} The desktop shell is built using QtQuick, a collection of technologies that are designed to help developers create intuitive, modern, and fluid user interfaces. It includes QML, a CSS & JavaScript like language to ease the development of dynamic and modern applications. {% endblocktrans %}

{% blocktrans %} QtQuick applications rendering is fully hardware accelerated and designed to run at 60Hz and is synchronized with VSync in order to avoid tearing, lad and flickering. {% endblocktrans %}

{% blocktrans %} Qt Creator is a cross-platform IDE (integrated development environment) tailored to the needs of Qt developers. It focuses on providing features that help new Qt users get up and running faster, and also boost the productivity of experienced Qt developers. {% endblocktrans %}
